TO-DO BLAST! — README

TO-DO BLAST! is a funky, animated, single-page To-Do application built with plain HTML, CSS, and JavaScript.
Perfect for learning DOM manipulation, localStorage, theme toggling and shipping a polished small project to your GitHub portfolio.

Features

Add, edit and remove tasks
Mark tasks as completed (checkbox)
Filter tasks: All, Active, Done
Persistent storage using localStorage (tasks survive page refresh)
Theme toggle (Light / Dark) stored in localStorage
Animated background particles and smooth UI interactions
Stats panel: Total, Pending, Crushed
“Nuke All Tasks” (clear all) with confirmation
Keyboard shortcuts:
Ctrl/Cmd + K — focus input
Ctrl/Cmd + D — toggle theme
